使用方法

1

安裝前置條件

安裝 Python 3.11 或更高版本，然後使用 Poetry 安裝依賴：

poetry install
2

配置 Claude Desktop

將伺服器添加到 claude_desktop_config.json 文件中：

{
    "mcpServers": {
        "web-search-duckduckgo": {
            "command": "poetry",
            "args": [
                "run",
                "python",
                "/path/to/web-search-duckduckgo/main.py"
            ]
        }
    }
}
3

使用工具

在 MCP 客戶端（例如 Claude）中使用以下工具：

  • search_and_fetch：搜尋網路並獲取 URL 內容
  • search：僅搜尋網路，不獲取頁面內容
  • fetch：獲取特定 URL 的內容
  • suggest_related_queries：提供相關查詢建議
  • get_cache_statistics：獲取快取統計資訊
  • clear_cache：清除所有快取資料
  • set_search_preferences：設定搜尋偏好
  • system_monitor：監控系統資源使用情況
  • health_check：檢查系統各組件是否正常運行

系統需求

  • Python 3.11+
  • 依賴套件 (透過 Poetry 管理)：
    • beautifulsoup4
    • httpx
    • mcp[cli]
    • python-dotenv
    • cachetools
    • psutil (用於系統監控功能)
    • requests
