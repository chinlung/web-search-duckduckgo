DuckDuckGo Web 搜尋工具

強大的網路搜尋與內容獲取工具，基於 DuckDuckGo 搜尋引擎

增強您的搜尋體驗

使用我們的 MCP 伺服器，輕鬆搜尋網路並獲取內容，支援快取管理和高級搜尋功能

功能特點

基本搜尋功能

  • 使用 DuckDuckGo 搜尋網路
  • 提取標題、URL 和摘要
  • 獲取 URL 內容並轉換為 Markdown
  • 並行獲取多個 URL

  • 提供快取使用情況統計
  • 清除快取資料
  • 智能快取時間設定

  • 提供相關查詢建議 (需自行設定開啟)
  • 自訂搜尋參數 (需自行設定開啟)
  • 查看當前設定
  • 支援多種搜尋區域

高級搜尋功能

  • 支援過濾和排序
  • 按域名、關鍵字過濾
  • 按標題、日期排序
  • 提供搜尋結果分析和摘要

使用方法

1

安裝前置條件

安裝 Python 3.11 或更高版本，然後使用 Poetry 安裝依賴：

poetry install
2

配置 Claude Desktop

將伺服器添加到 claude_desktop_config.json 文件中：

{
    "mcpServers": {
        "web-search-duckduckgo": {
            "command": "poetry",
            "args": [
                "run",
                "python",
                "/path/to/web-search-duckduckgo/main.py"
            ]
        }
    }
}
3

使用工具

在 MCP 客戶端（例如 Claude）中使用以下工具：

  • search_and_fetch：搜尋網路並獲取 URL 內容
  • search：僅搜尋網路，不獲取頁面內容
  • fetch：獲取特定 URL 的內容
  • suggest_related_queries (需自行設定開啟)：提供相關查詢建議
  • get_cache_statistics (需自行設定開啟)：獲取快取統計資訊
  • clear_cache (需自行設定開啟)：清除所有快取資料
  • set_search_preferences (需自行設定開啟)：設定搜尋偏好

注意：標記為「需自行設定開啟」的功能在 main.py 中已經實現但被註釋掉，需要您手動取消註釋才能使用。
